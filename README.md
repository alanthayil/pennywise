# ⚡ Pennywise - Expense Tracker

A simple, beautiful, browser-based expense tracker. No signups, no servers, no database. Your data stays in your browser.

## Features

- Add income & expenses with description, amount, category, and date
- Visual doughnut chart showing spending breakdown
- Filter transactions (All / Income / Expenses)
- Export to CSV
- Data persists in localStorage
- Works entirely offline after first load

## How to use

1. Open `index.html` in your browser (or serve via `python3 -m http.server`)
2. Start adding transactions
3. That's it.

## Tech

Pure HTML + CSS + JavaScript. Chart.js for visualizations. localStorage for data.
